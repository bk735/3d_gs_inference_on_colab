The repo contains:-
 -3d_gs.ipynb:- For those who want to train and render a scene using 3D_Gaussian Splatting. Here we have trained the dataset on all images as well as on sparse scenery. Helpful for baseline with all metrics demonstrated.
 -comparison_test_video:- The video is a comparison of a scene trained on 169 images and tested on 25 images meanwhile the right side demonstrate a scene trained on 7 evenly spaced image and tested on 25 images.
 -point_cloud_image_and_sparse_image:- These 2 images are rendered images using our output point clouds.They have been rendered on supersplat
